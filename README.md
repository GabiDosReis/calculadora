# 🧮 Calculadora em Python

Este projeto é uma calculadora desenvolvida em Python, utilizando conceitos de programação modular e armazenamento de histórico.

---

## 📌 Funcionalidades

* ➕ Soma
* ➖ Subtração
* ✖️ Multiplicação
* ➗ Divisão
* 📊 Porcentagem
* √ Raiz quadrada
* ⬜ Potência ao quadrado (^2)
* 🧾 Histórico de operações

---

## 🧱 Estrutura do Projeto

```
calculadora/
│
├── main.py          # Programa principal
├── operacoes.py     # Biblioteca com funções matemáticas
└── README.md        # Documentação do projeto
```

---

## ⚙️ Como executar o projeto

1. Abra o terminal na pasta do projeto
2. Execute o comando:

```
python main.py
```

---

## 💻 Exemplo de uso

```
Selecione: 1-calcular, 2-histórico, 3-finalizar
```

---

## 🧠 Conceitos utilizados

* Funções modulares (criação de biblioteca própria)
* Estruturas de repetição (`while`)
* Estruturas condicionais (`if/elif`)
* Listas (armazenamento de histórico em matriz)
* Entrada de dados com `input()`

---

## 📈 Exemplo de histórico

```
[10, '+', 5, '=', 15]
[9, 'raiz', '-', '=', 3]
[4, '^2', '-', '=', 16]
```
